# Slack UI Spec — Secretary M1 리디자인 레퍼런스

CDP로 실제 Slack 웹앱(app.slack.com)에서 추출한 정확한 스펙.
스크린샷: /tmp/slack_01_main.png ~ /tmp/slack_06_composer.png

## 전체 레이아웃

```
┌──────┬───────────┬──────────────────────────────┐
│ Icon │ Sidebar   │  Channel Header (49px)       │
│ Rail │ (289px)   ├──────────────────────────────┤
│(70px)│           │                              │
│      │ - 허들    │  Messages                    │
│ 홈   │ - 디렉터리│  (스크롤 영역)               │
│ DM   │ - 즐겨찾기│                              │
│ 활동 │ - 채널    │                              │
│ 파일 │   # 일반  │                              │
│ ...  │   # 개발  │                              │
│      │ - DM     │                              │
│ 관리 │   @ user  ├──────────────────────────────┤
│      │ - 앱     │  Composer (141px)             │
│ +    │   Slack..│  [툴바] [입력창] [전송]       │
│ 프로필│          │                              │
```

### 핵심 색상 팔레트
```css
--slack-bg-primary: #1a1d21;      /* 메인 배경 */
--slack-bg-secondary: #222529;    /* 입력창 등 약간 밝은 배경 */
--slack-bg-sidebar: rgba(16, 18, 20, 0.6); /* 사이드바 (반투명) */
--slack-text-primary: #d1d2d3;    /* 기본 텍스트 */
--slack-text-bright: #f8f8f8;     /* 강조 텍스트 (발신자 이름 등) */
--slack-text-muted: #ababad;      /* 타임스탬프 등 보조 텍스트 */
--slack-text-dark: #0c0f0f;       /* 활성 채널 텍스트 (밝은 배경 위) */
--slack-border: rgb(86, 88, 86);  /* 입력창 테두리 */
--slack-active-bg: #f8f8f8;       /* 선택된 채널 배경 */
```

## Icon Rail (좌측 70px)

### 구조
- 워크스페이스 아이콘 (36x36, 최상단)
- 구분선
- 탭 버튼들 (52x68 각각, 아이콘+라벨 수직 배치)
  - 홈 (selected)
  - DM
  - 내 활동
  - 파일
  - 더 보기...
  - (구분선)
  - 관리자
- 하단: + 버튼, 프로필 아바타

### 스타일
- 버튼 크기: 36x36 (아이콘 자체), 52x68 (라벨 포함 영역)
- 간격: gap 12px
- 패딩: top 8px
- 아이콘 색상: #f8f8f8
- borderRadius: 0px (사각형)

## Sidebar (289px)

### 구조 (위→아래)
1. **워크스페이스 헤더** (49px) — "Geneva" + 설정/글쓰기 아이콘
2. **허들** (28px)
3. **디렉터리** (28px)
4. **즐겨찾기** 섹션 — 접기/펼치기 가능
5. **채널** 섹션
   - # 새-워크스페이스-전체 (활성)
   - # 새-채널
   - # 소셜
6. **다이렉트 메시지** 섹션
   - geneva (나)
7. **앱** 섹션
   - Slackbot (배지: 1)

### 아이템 스타일
- 높이: 28px
- 패딩: 0 8px 0 24px (들여쓰기)
- borderRadius: 6px
- 기본: 투명 배경, #d1d2d3 텍스트
- 활성: #f8f8f8 배경, #0c0f0f 텍스트 (반전)
- hover: 별도 (캡처 미진행)

## Channel Header (49px)

### 구조 (좌→우)
- ☆ 즐겨찾기 토글
- **# 새-워크스페이스-전체** (채널 이름, bold)
- (중앙 여백)
- [팀원 초대하기] 버튼
- [허들] 아이콘
- [북마크] 아이콘
- [검색] 아이콘
- [⋮] 더보기

### 탭 바 (헤더 아래)
- 메시지 (활성, 밑줄)
- 캔버스 추가
- +

## Messages

### 메시지 구조
```
[아바타 36px]  [발신자 이름 (bold 900)]  [시간 12px #ababad]
               [메시지 본문 15px]
```

### 스타일
- 아바타: 36x36px
- 발신자 이름: 15px, weight 900, #f8f8f8
- 타임스탬프: 12px, #ababad
- 본문: 15px, line-height 22px, #d1d2d3
- 메시지 최소 높이: ~52px

## Composer (입력창)

### 구조
```
┌─────────────────────────────────────────────┐
│ [B] [I] [U] [S] [🔗] [≡] [≡≡] [<>] [AI]  │  ← 서식 툴바
│                                             │
│ #채널명에 메시지 보내기                       │  ← placeholder
│                                             │
│ [+첨부] [Aa서식] [😊] [@] [📹] [🎙] [⚡]  [▶전송] │  ← 하단 툴바
└─────────────────────────────────────────────┘
```

### 스타일
- 높이: 117px (기본, 늘어남 가능)
- 배경: #222529
- 테두리: 0.67px solid rgb(86, 88, 86)
- borderRadius: 8px
- 입력 텍스트: 15px, line-height 22px, #d1d2d3

### 하단 툴바 아이콘 (좌→우)
1. 첨부 (+)
2. 서식 표시 (Aa)
3. 이모티콘 (😊)
4. 멘션 (@)
5. 비디오 클립 (📹)
6. 오디오 클립 (🎙)
7. 단축키 (⚡)
8. --- (우측 끝) ---
9. 지금 보내기 (▶)
10. 나중에 예약 (⏱)

## Secretary 적용 시 매핑

| Slack 요소 | Secretary 대응 |
|-----------|---------------|
| Icon Rail | 메인 네비: 대시보드, 대화, AI대화, 할일, 시간, 설정 |
| Sidebar 채널 | 카테고리별 분류 (업무, 개발, 건강 등) |
| Channel Header | 선택된 카테고리/뷰 이름 + 검색 |
| Messages | 메시지 타임라인 (텔레그램 + AI CLI 통합) |
| Composer | 웹 입력창 (Secretary에 직접 입력) |
| DM 섹션 | AI 프로바이더별 (Claude, Codex, Gemini) |
