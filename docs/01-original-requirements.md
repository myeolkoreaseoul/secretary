## 추가 요구사항 (대화 과정에서 확인)

### 4. 시간 추적 + Daily Report 자동화

> 난 내가 매 시간 매일을 어떻게 쓰는지 추적해서, 낭비하는 시간을 없애고 싶어.

**Daily Report 형식 (3열 구조):**

| 시간 | 계획 (자동) | 실행 (자동+수동) | 몰입도 (AI+수동) |
|------|------------|-----------------|-----------------|
| 0시 | 수면 | 수면 | - |
| 9시 | 개발 공부 | VS Code 2시간 | ★★★★★ |
| 10시 | 개발 공부 | Chrome: 강의 | ★★★★☆ |
| 11시 | 소개팅앱 기획 | 카카오톡+유튜브 | ★★☆☆☆ |
| ... | ... | ... | ... |

**자동화 요구사항:**
- **계획 열**: 할일 리스트에서 우선순위대로 시간표에 자동 배치
- **실행 열 (온라인)**: PC 활동 자동 추적 (1분 간격, 활성 윈도우 기록)
- **실행 열 (오프라인)**: 수동 기록 (잠, 식사, 운동, 이동 — 대부분 시간 고정)
- **몰입도**: AI가 활동 내용 보고 자동 판단 + 오프라인은 수동
- **스마트폰**: 1단계에서는 수동 기록 (네이티브 앱 필요해서 어려움)

**활동 추적 원칙:**
1. 모든 활동은 온라인과 오프라인
2. 온라인 활동은 PC와 스마트폰에서 이루어짐
3. PC 온라인 활동은 자동 추적 가능 (WSL → PowerShell → 활성 윈도우)
4. 오프라인 활동은 수동 기록 (대부분 시간 고정)
